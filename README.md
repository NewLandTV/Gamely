# 소개

Gamely는 Game + Simply로, 게임을 더 간단하게 접근하도록 도와주는 웹사이트입니다.

## 사이트 목표

다양한 게임을 통해서 모든 게임을 빠르고 쉽게 접할 수 있도록 도와주는 사이트.

## 사이트 설명

여러 게임을 즐기고 싶은 사람들이 이 사이트에 들어와 검색 또는 인기게임을 간편하게 즐길 수 있도록 한 무료 종합 게임 사이트
